# sudoko
本次实验完成了基于小数的数独终盘的生成和基于大数的数独初盘生成，同时实现了DLX算法求解数独问题。本质上来说，三者都是通过回溯方法进行搜索，在搜索过程中采用了适当的启发式方法，减少了搜索次数。DLX算法则是通过构建合适的数据结构，模仿了人做数独的候选数方法，优化了回溯过程，提高了求解效率。
